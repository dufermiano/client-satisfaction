Tenha o [Docker](https://www.docker.com/) instalado na sua máquina

## Executar Aplicação

Você deve ter o Docker instalado na sua máquina, pois já vai subir a base de dados junto e rodar o seguinte comando:

```
docker-compose up
```

Para encerrá-la:

```
docker-compose down
```
